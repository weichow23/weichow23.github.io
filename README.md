# Wei Chow Personal Homepage

This repository contains a lightweight static GitHub Pages site for
[weichow23.github.io](https://weichow23.github.io).

The site is intentionally plain HTML and CSS so it can be published directly by
GitHub Pages without Ruby, Node, Jekyll themes, generated notebooks, or build
steps.

## Structure

- `index.html` - homepage with profile, publications, projects, and service
- `gallery/index.html` - photography gallery
- `assets/styles.css` - shared responsive styling
- `images/` - images used by the live pages
